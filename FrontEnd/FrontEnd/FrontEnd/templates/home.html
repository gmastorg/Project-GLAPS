Who Are We.

This project is comprised of 4 team members: 2 working on the front end, and 2 working on the back end. We are a strong group of students from Fayetteville Technical Community College who are applying knowledge obtained from the past two years to create a new innovative product. Through this, we have learned, in order for a project to be successful the team must work as a family; using each others strengths to help one another learn and surpass weaknesses.

Our Process.

Due to the scale of the project each team began by working on their corresponding section. The Front-End team divided each section of the site into more manageable parts that each team member could complete. The Back-End team began with data collection through the use of the census API. Once the data was collected, it was put into various vitalizations to note any errors, anomalies, or missing values. Next, the machine learning model was created and trained. Finally, both teams came together and created a bridge via a call to the Back-End API to give the Front-End the functionality and use of the machine learning algorithm.

Our Approach.

GLAPS was creates in two parts. There is this page which provides a user friendly interaction to the machine learning algorithm that creates the property value predictions. Users can easily select varying counties through the use of the map and enter the specific value of the home for which they would like to receive a prediction. The site then contacts the API that was created to directly interact with the machine learning algorithm.

Our Goal.

GLAPS' overall goal is to be able to predict the long term effects of varying developmental projects (parks, malls, stadiums, etc.) on property values. In this iteration, we were able to predict the effects of adding or removing professional baseball stadiums on property values using the 2017 census data.

Our Mission.

Provide an accurate prediction of property values to users via a service that is easy to use and comprehend. Thus, empowering them to make informed decisions on varying developmental projects proposed in their area.
